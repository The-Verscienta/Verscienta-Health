### 3. Rate Limiting & Security Middleware

**File Created**:

- `apps/web/middleware.ts`

**Features Implemented**:

- ‚úÖ Per-endpoint rate limiting configuration
- ‚úÖ Authentication endpoints: 5 requests / 15 min (brute force protection)
- ‚úÖ API endpoints: 10-100 requests / minute
- ‚úÖ Client identification via IP (x-forwarded-for, x-real-ip)
- ‚úÖ Automatic cleanup of expired rate limit entries
- ‚úÖ Rate limit headers (X-RateLimit-Limit, X-RateLimit-Remaining, X-RateLimit-Reset)

**Rate Limit Configuration**:

```typescript
'/api/auth/login': { requests: 5, window: 15 * 60 * 1000 }    // 5 req / 15 min
'/api/auth/register': { requests: 3, window: 60 * 60 * 1000 } // 3 req / 1 hour
'/api/grok': { requests: 10, window: 60 * 1000 }              // 10 req / 1 min
'/api': { requests: 100, window: 60 * 1000 }                  // 100 req / 1 min
default: { requests: 300, window: 60 * 1000 }                 // 300 req / 1 min
```

**Security Headers Implemented**:

- ‚úÖ Content-Security-Policy (CSP) with strict directives
- ‚úÖ Strict-Transport-Security (HSTS) with preload
- ‚úÖ X-Content-Type-Options: nosniff
- ‚úÖ X-Frame-Options: DENY (clickjacking prevention)
- ‚úÖ X-XSS-Protection: 1; mode=block
- ‚úÖ Referrer-Policy: strict-origin-when-cross-origin
- ‚úÖ Permissions-Policy (camera, microphone, geolocation restrictions)
- ‚úÖ Removed X-Powered-By header (version obfuscation)

**CSP Directives**:

```typescript
'default-src': ["'self'"]
'script-src': ["'self'", "'unsafe-inline'", "https://cdn.jsdelivr.net", "https://unpkg.com"]
'style-src': ["'self'", "'unsafe-inline'", "https://fonts.googleapis.com"]
'img-src': ["'self'", "data:", "blob:", "https:"]
'connect-src': ["'self'", CMS_URL, "https://api.openai.com", "https://*.algolia.net"]
'frame-ancestors': ["'none'"]  // Prevent embedding
'upgrade-insecure-requests': []
```

**Production Note**: In-memory rate limiting should be replaced with Redis for distributed deployments.

**Usage Example**:

```typescript

    }
  })
```

### Alerting Thresholds

**Critical Alerts** (immediate notification):

- 10+ failed logins from same IP in 5 minutes
- PHI access outside business hours (for certain roles)
- Audit logging failure
- Database encryption disabled
- Suspicious data export (>1000 records)

**Warning Alerts** (24-hour review):

- 5+ failed logins from same IP
- Multiple users from same IP
- Rate limit exceeded 10+ times
- Unusual geographic access patterns

**Info Alerts** (weekly review):

- New user registrations
- Role changes
- Configuration modifications
- Backup completion status
